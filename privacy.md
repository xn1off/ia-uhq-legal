# Politique de confidentialité - IA-UHQ

IA-UHQ peut traiter les informations nécessaires à son fonctionnement sur Discord.

## Données collectées
- Identifiants Discord (ID utilisateur, ID serveur, ID salon).
- Messages uniquement si une fonctionnalité du bot en a besoin.

## Utilisation
Les données sont utilisées uniquement pour fournir les fonctionnalités du bot.

## Partage
Les données ne sont ni vendues ni partagées avec des tiers.

## Suppression
Vous pouvez demander la suppression de vos données en contactant le développeur.
